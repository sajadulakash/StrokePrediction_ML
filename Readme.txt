Step 01: Download the Zip file and unzip the file 
Step 02: Install python 
Step 03: Install VS code and open the project folder using VS code
Step 04: open the terminal of vs code and run the given commands
Step 05: Install the necessary libraries-
	-pip install flask
	-pip install joblib
	-pip install numpy
	-pip install scikit-learn 
	-pip install -U scikit-learn==1.3.2
Step 06: open app.py and go to the line number 29. After that change the model path according your project folder location. remember you need to use "/" rather than "\".
Step 07: Run the app.py file
Step 08: open the local server (http://127.0.0.1:7384)
